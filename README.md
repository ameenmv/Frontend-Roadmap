## Foundation

### Understanding the Web

Frontend development is the practice of building user interfaces using HTML, CSS, and JavaScript. Everything users see and interact with—navigation, buttons, forms, animations—falls under frontend development.

**Essential Concepts:**
- How the Internet works
- HTTP protocol fundamentals
- DNS and domain names
- Browser architecture and rendering
- Client-server communication

**Core Resources:**
- [Frontend Development Overview](https://frontendmasters.com/guides/learning-roadmap/front-end-development-overview/)
- [How the Internet Works - MDN](https://developer.mozilla.org/en-US/docs/Learn/Common_questions/How_does_the_Internet_work)
- [HTTP Overview](https://developer.mozilla.org/en-US/docs/Web/HTTP/Overview)
- [Understanding Browsers](https://www.mozilla.org/en-US/firefox/browsers/what-is-a-browser/)

## Core Technologies

### HTML - Structure

HTML (Hypertext Markup Language) provides the semantic structure for web content. It defines meaning and accessibility for text, images, and multimedia.

**Key Topics:**
- Document structure and semantics
- Forms and input elements
- Accessibility best practices
- HTML5 APIs

### CSS - Styling & Layout

CSS (Cascading Style Sheets) controls visual presentation, layout, and responsive design. Modern CSS enables complex layouts and animations without JavaScript.

**Core Concepts:**
- Box model and positioning
- Flexbox and Grid layouts
- Responsive design principles
- CSS animations and transitions
- Modern CSS features (variables, calc, clamp)

**Comprehensive Resources:**

## Intermediate Skills

### JavaScript - Programming

JavaScript brings interactivity to web pages. Master both fundamental programming concepts and modern ES6+ features.

**Essential Topics:**
- Variables, data types, operators
- Functions and scope
- Arrays and objects
- DOM manipulation
- Event handling
- Asynchronous JavaScript (Promises, Async/Await)
- ES6+ features (arrow functions, destructuring, modules)
- Fetch API and AJAX
- Object-oriented programming
- Regular expressions

**Fundamental Resources:**
- [JavaScript Fundamentals](https://frontendmasters.com/guides/learning-roadmap/javascript-fundamentals/)

### JavaScript Frameworks

Modern frameworks provide structure, state management, and tooling for complex applications.

**Framework Benefits:**
- Component-based architecture
- Reactive data binding
- Routing solutions
- State management
- Server-side rendering
- Large ecosystems

**Top Frameworks:**
- **React** - Component library with vast ecosystem
- **Vue** - Progressive framework with gentle learning curve
- **Angular** - Full-featured enterprise framework
